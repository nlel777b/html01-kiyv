1 Tabelele HTML
  • Tabelele HTML sunt o modalitate de a prezenta informatii in structura tabelara de randuri si coloane
  • Elemente esentiale ale tabelelor sunt table, tr si td
  • https://developer.mozilla.org/en-US/docs/Learn/HTML/Tables/Basics

<. /tbody>
<.tfoot>
<.tr>
<.th> <. /th>
<. /tr>
<. /tfoot>
<. /table>
2 Elementul table
  • Elementul table este un element de tip block, semantic care marcheaza informatia tabelara
  • Elementul table poate contine exclusiv elemente de tip tr sau tbody
  • https://developer.mozilla.org/en-US/docs/Web/HTML/Element/table
 <.table>
<.tr>
<.td>Nume<. /td>
<.td>Exemple<. /td>
<.td>Detalii<. /td>

<.tr>
<.td>value1<. /td>
<.td>value2<. /td>
<.td>Atributul decide valoarea care va fi selectata<. /td>
<. /tr>
<. /table>
3 Elementul tr
  • Elementul tr (table row) este unitatea de rand ce compune un tabel
  • Acest element poate contine doar elemente td sau th
  • https://developer.mozilla.org/en-US/docs/Web/HTML/Element/tr
 <.table>
<.tr>
<.td>Nume<. /td>
<.td>Exemple<. /td>
<.td>Detalii<. /td>
<. /tr>
<.tr>
<.td>value1<. /td>
<.td>value2<. /td>
<.td>Atributul decide valoarea care va fi selectata<. /td>
<. /tr>
<. /table>
4 Elementul td
  • Elementul td este unitatea primara a tabelelor HTML reprezentand celula de baza
  • Elementul td poate contine orice element, inclusiv alte tabele
  • Poate fi interschimbat cu th
<.table>
<.tr>
<.td>Nume<. /td>
<.td>Exemple<. /td>
<.td>Detalii<. /td>
<. /tr>
<.tr>
<.td>value1<. /td>
<.td>value2<. /td>
<.td>Atributul decide valoarea care va fi selectata<. /td>
<. /tr>
<. /table>
5 Elementul th
  • Elementul th (table header cell) poate inlocui td atunci cand se marcheaza capetele de tabel
  • Se aplica regulile de la elementul td
  • Pot fi mai multe capete de tabel intr-un element de tip table
  • https://developer.mozilla.org/en-US/docs/Web/HTML/Element/th
 <.thead>
<.tr>
<.th>Exemplu<. /th>
<. /tr>
<. /thead>
